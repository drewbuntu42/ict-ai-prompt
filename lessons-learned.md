# Lessons Learned — ICT AI Prompt Competition

## Architecture Alignment Must Precede Team Parallelization

**What happened:** Team members built React components in parallel without a shared architecture contract. Integration became the hardest problem at the most time-critical moment of the competition — when there was no time left to resolve structural mismatches.

**The lesson:** Before splitting work across people, align on:
- Component interface contracts (props shape, data format, export names)
- State ownership (who owns what data, how it flows)
- Shared dependencies (which libraries, which versions)
- File/module structure (where things live, how they import each other)

Architectural alignment is not overhead — it is the prerequisite for parallelization to be safe. Without it, parallel work creates integration debt that compounds under time pressure.

**Rule:** Architecture spike first, then divide. Even 15 minutes of whiteboarding the component tree and data contracts before splitting tasks would have prevented the integration crunch.

---

## Prompting for Components Without an Integration Plan Produces Orphans

Components generated by AI in isolation — each prompted separately, each making its own layout and data assumptions — do not compose cleanly. The more capable the AI, the more confident each component is in its own conventions, and the harder the seams are to reconcile.

**Rule:** When using AI to generate multiple components that will be composed together, write the integration layer first (or at minimum, define it). Prompt for leaf components after the container contract exists.

---

## Time-Sensitive Deliverables Require a "Hard Stop" Checkpoint

Competition format compresses feedback loops. There is no natural pause between "build" and "ship." Without a deliberate checkpoint to verify integration before crunch, integration failures surface at the worst possible moment.

**Rule:** Set an explicit integration checkpoint at the midpoint of any timed deliverable — not at the end. If components don't compose cleanly at the midpoint, there is still time to adapt.
